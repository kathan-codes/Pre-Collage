# Day 2: News Headline Aggregator 📰

## 📌 Project Overview
A web scraping script that extracts daily headlines from a news website or tech blog, cleans up the text layout, and displays them cleanly in the terminal.

## 🎯 Learning Objectives
* Understand HTML structure, tags, classes, and IDs.
* Learn how to navigate a DOM (Document Object Model) parsing tree.
* Use CSS selectors to target and extract specific website elements.

## 🛠️ Core Libraries Used
* `requests` - To fetch the raw HTML content of the website.
* `beautifulsoup4` - To parse the HTML and extract the headline text.
