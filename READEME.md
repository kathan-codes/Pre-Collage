# Day 1: Crypto Price Alert 🪙

## 📌 Project Overview
A lightweight Python script that fetches real-time cryptocurrency data from a free public API and prints out clean, formatted price updates.

## 🎯 Learning Objectives
* Understand HTTP basic methods (`GET` requests).
* Learn how to parse nested `JSON` data structures in Python.
* Handle API responses using the `requests` library.

## 🛠️ Core Libraries Used
* `requests` - For handling the internet connection and API data retrieval.
