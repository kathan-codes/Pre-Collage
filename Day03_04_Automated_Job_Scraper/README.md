# Days 3 & 4: Automated Job Scraper 🤖💼

## 📌 Project Overview
A browser automation script that launches a live browser instance, navigates to a job board, interacts with search filters programmatically, and extracts information about hiring companies.

## 🎯 Learning Objectives
* Understand how to control a headless or live automated browser.
* Learn to interact with JavaScript-heavy dynamic websites.
* Simulate human behaviors such as button clicks, page scrolling, and form inputs.
* Locate web elements dynamically using XPath and CSS selectors.

## 🛠️ Core Libraries Used
* `selenium` - For driving the automated browser and handling dynamic web elements.
